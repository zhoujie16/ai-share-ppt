# Codex App 的功能用法｜PPT 大纲

## 制作原则

- 每个章节先放 1 页文字导入页，说明本章要讲什么、观众应该关注什么。
- 随后进入连续素材页。静态图片页的幻灯片画面只放 1 张图片，不叠加标题、说明文字或装饰元素。
- 图片页的标题和讲解点建议放在演讲备注里，用于现场讲解。
- 素材顺序按 `图片/` 目录中的数字前缀和自然排序排列；文件名只作为参考，章节组织以截图内容为主。
- 第 11 章末尾包含 1 个视频素材，单独安排为视频演示页。

## 开场页

### 第 1 页：封面

- 标题：Codex App 的功能用法
- 副标题：从一句话创建项目，到云端 PR、插件自动化、移动协作和跨应用操作
- 讲解重点：本次分享用一条完整工作流展示 Codex App 怎么介入真实工作：理解需求、执行任务、验证结果、进入工程流程。

### 第 2 页：分享目录

- 认识 Codex App 工作台
- 从一句话创建项目
- 用计划模式控制复杂任务
- 改造现有项目并验证
- 用分叉探索不同方案
- 提交代码
- 使用 Codex Web 云端开发与 PR
- 使用插件、技能与自动化
- 用 PPT 技能生成分享材料
- 部署上线
- 操作电脑完成跨应用任务
- 连接手机使用 Codex 移动版
